# Lab4 掷骰子APP实验报告
## 1. 界面结构说明
使用 Jetpack Compose 的 `Column` 布局，垂直居中排列骰子图片和「Roll」按钮，实现简洁直观的交互界面，符合题目要求的效果图。

## 2. 状态管理实现
使用 `remember { mutableStateOf(1) }` 存储骰子点数状态，点击按钮时更新状态，Compose 自动触发界面刷新，实现骰子图片的动态切换。

## 3. 核心逻辑说明
通过 `when` 表达式，根据 1-6 的随机点数，匹配对应 `dice_1` ~ `dice_6` 矢量图资源，完成骰子点数的随机切换，点击「Roll」按钮即可触发。

## 4. 调试操作说明
1.  在 `result = (1..6).random()` 行设置断点
2.  点击 Debug 按钮运行，点击「Roll」触发断点
3.  使用 `Step Over` 单步执行，观察 `result` 变量的随机变化，验证逻辑正确性

## 5. 调试功能体会
- `Step Into`：进入函数内部执行，可查看函数内部逻辑
- `Step Over`：逐行执行代码，不进入函数，适合快速排查
- `Step Out`：跳出当前函数，回到上一层调用位置

## 6. 遇到的问题与解决
1.  语法报错：第6行多余的import语句导致编译失败，通过全量替换完整代码解决
2.  图片资源问题：将骰子矢量图正确放入 `res/drawable` 文件夹，解决资源引用问题
3.  状态不更新：使用 `mutableStateOf` 管理状态，确保点击按钮时界面能响应刷新
